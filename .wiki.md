# Project Summary
Stackely is an AI-driven tool recommendation engine that assists users in discovering digital tools tailored to their specific business needs. Targeting freelancers, marketers, creators, e-commerce founders, and small business owners, it simplifies the software selection process through a comprehensive database, modern user interface, and advanced AI capabilities. Stackely enhances decision-making, enabling users to efficiently find the right tools for their projects.

# Project Module Description
The project consists of several functional modules:
- **Database**: Manages a collection of tools and user queries, including fields for `logo_url`, `tool_type`, and `visitor_id`.
- **AI Classification**: Analyzes user input, detects use cases, and incorporates a pricing preference step for tailored recommendations.
- **Frontend**: User interface for inputting goals and displaying recommended tools, redesigned for modern aesthetics and branding consistency.
- **Admin Dashboard**: Interface for managing tools and user queries, secured with a secret key.
- **Legal Pages**: Static pages for Privacy Policy, Terms of Service, and Cookie Policy, ensuring legal compliance and updated contact information.
- **Sitemap**: A dynamically generated `sitemap.xml` file for SEO, automatically updated to include all public pages.
- **Maintenance Mode**: A feature to display a branded maintenance page when the site is undergoing updates.
- **Robots.txt**: A file to guide search engine crawlers, now includes a reference to the sitemap.
- **Redirects**: A file to ensure static files are served directly, bypassing SPA fallback.

# Technology Stack
- **Backend**: Python, FastAPI
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Database**: SQLAlchemy for ORM and Alembic for migrations
- **AI**: DeepSeek for classification

# Usage
1. **Install Dependencies**: Navigate to the backend and frontend directories and run:
   ```
   pnpm install
   ```
2. **Build the Application**: In the frontend directory, run:
   ```
   pnpm run build
   ```
3. **Generate Sitemap**: To regenerate the sitemap, run:
   ```
   pnpm run generate:sitemap
   ```
4. **Run the Application**: Execute the start script:
   ```
   ./start_app_v2.sh
   ```
